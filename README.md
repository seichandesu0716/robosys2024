# kadai1コマンド
サッカーのPKを行うプログラム

# インストール方法 
**リポジトリの複製**
~~~
git clone https://github.com/seichandesu0716/robosys2024.git
~~~
**リポジトリの移動**
~~~
cd tmp/robosys2024
~~~
**実行**
~~~
./kadai1
~~~
